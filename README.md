# Bank Project - Procedural Version (C++)

## Overview
This is the **Procedural version** of the Bank Management System built in **C++**.  
The project is written using **functions only**, without classes or object-oriented concepts.  
It provides basic banking features such as adding clients, managing accounts, deposits, withdrawals, and saving data in text files.

---

## Features
- Manage clients (add, update, delete, list).
- Manage users (add, update, delete, list).
- Manage account balances (deposit, withdraw, transfer).
- Secure login system using user data stored in `Users.txt`.
- User access permissions to control which features each user can access.
- Data persistence using text files (`Clients.txt` & `Users.txt`).
